================================================================================
  DEMO WORKSHEET - QUICK REFERENCE
================================================================================

POPULATE SAMPLE DATA:
  Alt+F8 → PopulateSampleData → Run
  (Fills all sheets with test data)

CLEAR SAMPLE DATA:
  Alt+F8 → ClearSampleData → Run → Yes
  (Clears all test data)

================================================================================
  TESTING WORKFLOW
================================================================================

1. Load sample data (or a scenario)
2. Go to Position Sizing → Click "Calculate"
3. Go to Checklist → Click "Evaluate"
4. Go to Heat Check → Click "Check Heat"
5. Go to Trade Entry → Click "Save GO"
6. Clear sample data when done

================================================================================
  ADD BUTTONS TO DEMO SHEET (Optional)
================================================================================

1. Create sheet named "Demo"
2. Developer → Insert → Button (Form Control)
3. Draw button
4. Assign macro: PopulateSampleData (or other function)
5. Edit text to describe button
6. Repeat for other functions

Suggested buttons:
  - Populate Sample Data
  - Clear Sample Data
  - Scenario 1: Simple Stock
  - Scenario 2: Yellow Banner
  - Scenario 3: Option Trade

================================================================================
  TROUBLESHOOTING
================================================================================

Functions not in macro list?
  → Run fix-vba-modules.bat

Subscript out of range error?
  → Make sure sheets are named exactly:
    Position Sizing, Checklist, Heat Check, Trade Entry

Buttons don't work?
  → Enable macros (Enable Content button)
  → Right-click button → Assign Macro → Select function
